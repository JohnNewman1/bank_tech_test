Bank App

Open the Console to use App:



Balance will start at zero, to view the balance enter account.balance into console

To deposit funds enter the following command account.balance(amount, date). The date must be entered in the following format: DD-MM-YYYY

To withdraw funds enter the following command account.withdraw(amount, date). The date must be entered in the following format: DD-MM-YYYY

To print your transaction history enter the following command output.print().

To reset the app refresh the page
